## Data-mining

## Queries

Create "Select" queries, pivot-tables, summaries and multi-dimensional queries using the editor dialog:

![](https://github.com/Steema/TeeBI/blob/master/docs/img/TeeBI_Pivot_Query_Editor.png)

And at design-time, edit the properties and callbacks for each part of the query, if desired:

![](https://github.com/Steema/TeeBI/blob/master/docs/img/TeeBI_Query_Editor.png)

Geographical queries (ie, "sum by country") and charts are easy using the included "Geo" database:

![](https://github.com/Steema/TeeBI/blob/master/docs/img/TeeGrid_overview.png)
![](https://github.com/Steema/TeeBI/blob/master/docs/img/geo_database.png)

## Summaries

Dimensions of a summary include expressions (sum, average, etc), or a custom Delphi function callback:

![](https://github.com/Steema/TeeBI/blob/master/docs/img/summary_automatic_layout.png)

Also Histograms (numeric and alphabetic) to split results in groups by a given field:

![](https://github.com/Steema/TeeBI/blob/master/docs/img/summary_histogram.png)

## Gridifying

![](https://github.com/Steema/TeeBI/blob/master/docs/img/TeeBI_Gridify.png)

![](https://github.com/Steema/TeeBI/blob/master/docs/img/TeeBI_Gridify_colored.png)
